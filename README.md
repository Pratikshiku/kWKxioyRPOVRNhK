### 3. Functional Description
The cinema ticket system implemented with SpringBoot+Vue is designed with two roles: users and administrators.

**Regular User Functions:**
1. Login
2. Register
3. Movie Search
4. Ticket Purchase
5. Seat Selection
6. Movie Rankings
7. Coming Soon
8. Popular Movies


**Administrator Functions:**
Administrators can perform system management, cinema management, movie management, hall management, order management, user management, and role permission management

Cinema Management: Cinema Information Management: Can manage cinema name, opening hours, hall types, descriptions, etc.

**Movie Management**

Movie Information Management: Add, modify, search, delete movies, and manage movie types

Movie Category Management: Can add, modify, and delete movie categories

**Hall Management**

Hall Information Management: Can add, modify, delete cinema halls, manage seats, set seat availability, and search by name and category

Showtime Information Management: Can add, modify, delete movie showtimes in halls, and view seat booking status for each showtime

**Order Management**

Order Information Management: Manage user orders

**User Management**

User Information Management: Manage all users in the cinema system

**Role Permission Management**

Role Information Management, Permission Information Management
